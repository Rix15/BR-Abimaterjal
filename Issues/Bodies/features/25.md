**Journal Table**
https://learn.microsoft.com/en-us/training/modules/identify-table-types/8-create-journal-tables

Rendižurnaali rida

- BaseApp analoogiana saab kasutada kaubažurnaali rea tabeli objekti
- table 83 "Item Journal Line"

Tabeli väljad
- "Line No."
  - Integer, PK
- "Entry Type"
  - Enum "Rental Ledger Entry Type"
- "Document No."
- "Posting Date"
- "Rental Order No."
  - FK Rental Order
- "Customer No."
  - FK Customer
  - Täidetakse "Rental Order No." valideerimisel
- "Rental Asset No."
  - FK Rental Asset
  - Täidetakse "Rental Order No." valideerimisel
- "Rental Asset Unit No."
  - FK "Rental Asset Unit", kus "Rental Asset No." = "Rental Asset No."
  - Täidetakse "Rental Order No." valideerimisel
- Rental Start
  - Täidetakse "Rental Order No." valideerimisel
  - Valideerimisel arvutatakse Rental Days
- Rental End
  - Täidetakse "Rental Order No." valideerimisel
  - Valideerimisel arvutatakse Rental Days
- Rental Days
  - Täidetakse "Rental Order No." valideerimisel
  - Valideerimisel arvutatakse End Date


Funktsioon (kasutatakse konteerimisel)

IsEmptyLine
- Jooksva kirje (Rec) funktsioon
- Tagastab TRUE, kui olulised väljad on tühjad
  - "Rental Order No."
  - "Rental Start"
  - "Rental End"
  - "Rental Days"
