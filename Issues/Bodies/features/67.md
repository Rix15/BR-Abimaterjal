https://learn.microsoft.com/en-us/training/modules/build-power-bi-report/3-add-data-dimension

Lisa tabel Date

```
Date =
    ADDCOLUMNS (
        CALENDARAUTO(),
        "DateAsInteger", FORMAT ( [Date], "YYYYMMDD" ),
        "Year", YEAR ( [Date] ),
        "Monthnumber", FORMAT ( [Date], "MM" ),
        "MonthName", FORMAT ( [Date], "mmmm" ),
        "DayOfWeekNumber", WEEKDAY ( [Date] ),
        "DayOfWeek", FORMAT ( [Date], "dddd" ),
        "Quarter", "Q" & FORMAT ( [Date], "Q" )
    )
```
 
Loo hierarhia

- Year
- Quarter
- Monthnumber

Määra kuupäevatabeliks

- To set the date table, select the table you want to use as a date table in the Fields pane, then right-click the table and select Mark as date table > Mark as date table in the menu that appears.
